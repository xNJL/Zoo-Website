# README

### Install Requirements

pip install -r requirements.txt

### MySQL Setup

##### On Mac

From terminal:

1) /usr/local/mysql/bin/mysql -u root -p
2) CREATE USER 'test'@'localhost' IDENTIFIED BY 'test';
3) GRANT ALL PRIVILEGES ON * . * TO 'test'@'localhost';

##### On Ubuntu

From terminal:

1) mysql -u root -p
2) CREATE USER 'test'@'localhost' IDENTIFIED BY 'test';
3) GRANT ALL PRIVILEGES ON * . * TO 'test'@'localhost';
